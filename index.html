    How to Play

    • Unscramble each set of letters to form valid 4-letter words.
    • Type 'x' if you think the letter set cannot form a valid word.
    • For multiple valid words, separate them with spaces or commas.
    • You have 2 minutes to solve as many as you can.

    Scoring

    • +10 points for a fully correct answer or correctly identifying a fake with 'x'.
    • Partial points are awarded for partially correct answers.
    • -5 points for submitting any incorrect words.
    • -5 points for incorrectly marking a real word set as 'x'.
    • -25 points for using the extra time button.
